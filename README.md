# Flutter - Tic Tac Toe Multiplayer Game

## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/tictactoe-multiplayer-game
```

Go to the project directory

```bash
  cd tictactoe-multiplayer-game
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
